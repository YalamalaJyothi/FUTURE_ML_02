# Support Ticket Classification

## Objective
Classify customer support tickets into categories using Machine Learning.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Google Colab

## Features
- Text Classification
- Ticket Categorization
- Model Evaluation

## Project Description
This project uses Machine Learning and Natural Language Processing (NLP) to automatically classify customer support tickets into different categories such as Payment, Account, and Technical issues.
